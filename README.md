# SaaS Backend (Go)

## 🚀 Overview
A scalable backend system built using Go for handling SaaS applications.

## ⚙️ Tech Stack
- Go (Golang)
- REST APIs
- Modular Architecture

## 📦 Features
- User Authentication (JWT)
- API Routing
- Scalable Project Structure

## 📁 Project Structure
cmd/ → entry point  
internal/ → business logic  
pkg/ → reusable packages  

## ▶️ Run Locally
go run cmd/main.go
